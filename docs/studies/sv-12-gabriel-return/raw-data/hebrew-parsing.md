# Raw Hebrew Parsing Data

## Dan 8:16 — Gabriel's Commission
```
וָאֶשְׁמַע קוֹל־אָדָם בֵּין אוּלָי וַיִּקְרָא וַיֹּאמַר גַּבְרִיאֵל הָבֵן לְהַלָּז אֶת־הַמַּרְאֶה

Key morphology:
- הָבֵן = Verb.Hiphil.Impv.2ms of biyn (H995) — "make understand!" (causative)
- לְהַלָּז = Prep + Art + DemPron "this one there" (Daniel as indirect object)
- אֶת־הַמַּרְאֶה = Object marker + Art + mar'eh (H4758) — "the vision/sight" (direct object)
```

## Dan 8:27 — Failure Report
```
וַאֲנִי דָנִיֵּאל נִהְיֵיתִי וְנֶחֱלֵיתִי יָמִים ... וָאֶשְׁתּוֹמֵם עַל־הַמַּרְאֶה וְאֵין מֵבִין

Key morphology:
- נִהְיֵיתִי = Verb.Niphal.Perf.1s of hayah — "I became [faint]"
- נֶחֱלֵיתִי = Verb.Niphal.Perf.1s of chalah — "I became sick"
- אֶשְׁתּוֹמֵם = Verb.Hithpael.Wayq.1s of shamam — "I was astonished/desolate"
- עַל־הַמַּרְאֶה = "at/concerning the mar'eh" (same word as 8:16)
- וְאֵין מֵבִין = "and none understanding" — Hiphil ptcp of biyn (negated). The SAME root and SAME stem as the 8:16 commission. Commission: haben (Hiphil impv). Failure: ein mebiyn (Hiphil ptcp, negated).
```

## Dan 9:2 — Daniel's Independent Study
```
אֲנִי דָּנִיֵּאל בִּינֹתִי בַּסְּפָרִים מִסְפַּר הַשָּׁנִים

Key morphology:
- בִּינֹתִי = Verb.Qal.Perf.1s of biyn (H995) — "I understood" (SELF-GENERATED, Qal not Hiphil)
- בַּסְּפָרִים = "by/in the books" — instrumental bet; text-mediated understanding
- Contrast: 8:16 Hiphil (causative, angel-mediated) vs. 9:2 Qal (basic, self-generated)
```

## Dan 9:21 — Gabriel's Return
```
וְהָאִישׁ גַּבְרִיאֵל אֲשֶׁר רָאִיתִי בֶחָזוֹן בַּתְּחִלָּה מֻעָף בִּיעָף נֹגֵעַ אֵלַי כְּעֵת מִנְחַת־עָרֶב

Key morphology:
- הָאִישׁ גַּבְרִיאֵל = "the man Gabriel" — definite, named
- אֲשֶׁר רָאִיתִי = "whom I had seen" (Qal perf 1cs of ra'ah)
- בֶחָזוֹן = "in the chazon" — back-reference uses chazon (whole vision)
- בַּתְּחִלָּה = "at the beginning" (techillah H8462) — temporal pointer to Dan 8
- מֻעָף = Verb.Hophal.Ptcp of ya'aph — "caused to fly swiftly" (passive causative)
- כְּעֵת מִנְחַת־עָרֶב = "about the time of the evening oblation" — construct chain: minchath (H4503, fs.Cst) + ereb (H6153)
```

## Dan 9:22-23 — Gabriel's Resumption
```
v.22: וַיָּבֶן וַיְדַבֵּר עִמִּי ... עַתָּה יָצָאתִי לְהַשְׂכִּילְךָ בִינָה
v.23: ... וּבִין בַּדָּבָר וְהָבֵן בַּמַּרְאֶה

Key morphology:
- וַיָּבֶן = Verb.Hiphil.Wayq.3ms of biyn — "he caused to understand" (same stem as 8:16's commission)
- לְהַשְׂכִּילְךָ = Verb.Hiphil.InfCon of sakal (H7919) + 2ms suffix — "to give you prudential insight"
- בִינָה = Noun.fs.Abs — "understanding" (same word Daniel sought in 8:15)
- וּבִין = Verb.Qal.Impv.2ms of biyn — "understand!" (receptive)
- וְהָבֵן = Verb.Hiphil.Impv.2ms of biyn — "gain insight!" (active/causative)
- בַּמַּרְאֶה = "into the mar'eh" — SAME word, SAME article as 8:16's commission object
```

## Dan 9:24 — Seventy Weeks Decree
```
שָׁבֻעִים שִׁבְעִים נֶחְתַּךְ עַל־עַמְּךָ ... וּלְהָבִיא צֶדֶק עֹלָמִים וְלַחְתֹּם חָזוֹן וְנָבִיא וְלִמְשֹׁחַ קֹדֶשׁ קָדָשִׁים

Key morphology:
- נֶחְתַּךְ = Verb.Niphal.Perf.3ms of chathak (H2852) — "is cut off/decreed" (HAPAX)
- צֶדֶק עֹלָמִים = "righteousness of ages" (construct chain) — tsedeq (H6664) bridges to nitsdaq (8:14)
- לַחְתֹּם = Verb.Qal.InfCon of chatham (H2856) — "to seal up" (authentication mode)
- חָזוֹן = chazon (H2377) — "vision" (sealing the chazon, not the mar'eh)
- קֹדֶשׁ קָדָשִׁים = "holy of holies" / "most holy" — qodesh (H6944) bridges to 8:14
```

## Dan 8:26 — Dual Vision Terms
```
וּמַרְאֵה הָעֶרֶב וְהַבֹּקֶר אֲשֶׁר נֶאֱמַר אֱמֶת הוּא וְאַתָּה סְתֹם הֶחָזוֹן כִּי לְיָמִים רַבִּים

Key morphology:
- מַרְאֵה הָעֶרֶב וְהַבֹּקֶר = construct chain: "vision/sight of the evening and the morning" (mar'eh, articled absolutes)
- אֱמֶת הוּא = "truth [is] it" — verbless nominal clause authenticating the mar'eh
- סְתֹם = Verb.Qal.Impv.2ms of satham (H5640) — "shut up/seal"
- הֶחָזוֹן = "the chazon" — different word, different treatment (sealed, not authenticated)
- לְיָמִים רַבִּים = "for many days" — temporal prepositional phrase
```

## Dan 10:1 — Resolution
```
וּבִין אֶת־הַדָּבָר וּבִינָה לוֹ בַּמַּרְאֶה

Key morphology:
- וּבִין = Verb.Qal.Perf.3ms of biyn — "and he understood" (COMPLETED action)
- אֶת־הַדָּבָר = "the word/matter" (direct object)
- וּבִינָה = Noun.fs.Abs — "and understanding"
- לוֹ = "to him" (possessive dative)
- בַּמַּרְאֶה = "in/of the mar'eh" — SAME word as 8:16, 8:27, 9:23. The mar'eh is RESOLVED.
```

## Dan 10:14 — Chazon Continues
```
כִּי־עוֹד חָזוֹן לַיָּמִים

Key morphology:
- עוֹד = "yet, still" — duration continues
- חָזוֹן = chazon (H2377) — the broad vision, NOT mar'eh
- לַיָּמִים = "for the days" — prep le + art + yamim. Compare 8:26 le-yamim rabbim ("for many days"). Same prepositional frame but 10:14 drops rabbim.
```
