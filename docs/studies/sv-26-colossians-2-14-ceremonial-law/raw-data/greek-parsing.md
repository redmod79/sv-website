# Greek Parsing Raw Data

## Col 2:14
Greek: ἐξαλείψας τὸ καθ' ἡμῶν χειρόγραφον τοῖς δόγμασιν ὃ ἦν ὑπεναντίον ἡμῖν, καὶ αὐτὸ ἦρκεν ἐκ τοῦ μέσου, προσηλώσας αὐτὸ τῷ σταυρῷ·

Key parsing:
- ἐξαλείψας (G1813) Aor Act Ptcp Nom Sg M
- χειρόγραφον (G5498) Acc Sg N
- δόγμασιν (G1378) Dat Pl N
- ὑπεναντίον (G5227) Nom Sg N
- ἦρκεν (G142, airō) Perf Act Ind 3S — perfect tense = completed with ongoing result
- προσηλώσας (G4338) Aor Act Ptcp Nom Sg M

## Col 2:17
Greek: ἅ ἐστιν σκιὰ τῶν μελλόντων, τὸ δὲ σῶμα τοῦ Χριστοῦ.

Key parsing:
- ἅ (G3739) Neuter Pl — relative pronoun referring to ALL items in v.16
- σκιὰ (G4639) Nom Sg F — "a shadow"
- μελλόντων (G3195) Pres Act Ptcp Gen Pl N — "of the things coming"
- σῶμα (G4983) Nom Sg N — "the body/substance"
- Χριστοῦ (G5547) Gen Sg M — "of Christ"

## Heb 9:23
Greek: Ἀνάγκη οὖν τὰ μὲν ὑποδείγματα τῶν ἐν τοῖς οὐρανοῖς τούτοις καθαρίζεσθαι, αὐτὰ δὲ τὰ ἐπουράνια κρείττοσιν θυσίαις παρὰ ταύτας.

Key parsing:
- Ἀνάγκη (G318) Nom Sg F — "Necessity"
- ὑποδείγματα (G5262) Acc Pl N — "the patterns/copies"
- καθαρίζεσθαι (G2511) Pres Pass Inf — "to be purified"
- ἐπουράνια (G2032) Acc Pl N — "the heavenly things"
- κρείττοσιν θυσίαις (G2909, G2378) — "with better sacrifices"
