## Key Vocabulary Observations

### The Two-Axis Framework (Lev 10:10)
```
Axis 1: qodesh (H6944, holy) ←→ chol (H2455, common/profane)
Axis 2: tahor (H2889, clean)  ←→ tame (H2931, unclean)
```
These are independent axes. A thing can be:
- Holy + Clean (normal state of sanctuary objects)
- Holy + Unclean (sanctuary after defilement accumulates — the crisis state)
- Common + Clean (ordinary clean food, clean person)
- Common + Unclean (carcass, person with skin disease)

### Daily vs. Annual Vocabulary
- Daily: kaphar (H3722) + salach (H5545) = atonement + forgiveness
- Annual: kaphar (H3722) + taher (H2891) = atonement + cleansing
- salach is ABSENT from Lev 16
- taher is ABSENT from daily sin-offering passages (Lev 4)
- This vocabulary distinction confirms the two-phase system

### Three Defilement Vocabularies (Hebrew)
1. tame (H2930/2931/2932) — ritual/ceremonial impurity
2. chaneph (H2610) — moral pollution (land-defilement by bloodshed)
3. chalal (H2490) — profanation (treating holy as common)
